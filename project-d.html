Format

Website

Client

Bee Ineditus, Portugal
November 2018

Specificities

Website for the new brand of Portuguese children's clothing: 'Bee Child'.

The goal of this project is to promote children's clothing through a responsive website with a store.

Designed and developed with a WordPress template (chosen by the client) is a harmonious and romantic project that gives focus to the store and the children's clothing images.
